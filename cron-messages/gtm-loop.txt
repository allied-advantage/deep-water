### Your Mission
Create the full GTM launch package:

1. GOOGLE ADS
   - Research keywords for the opportunity
   - **Run Google Ads API forecast** if API access is available (credentials in `~/.openclaw/credentials/google-ads.json.enc`, MCC 960-044-1896)
   - Get estimated CPC, impressions, clicks at $1K/$3K/$5K monthly budgets
   - Write ad copy (headlines 30 chars, descriptions 90 chars)
   - Recommend campaign structure and budget backed by forecast data
   - Output to GTM_PLAN.md and gtm/google-ads/forecast.md

2. META ADS
   - Define target audiences (custom, lookalike, interest-based)
   - Write ad creative concepts (hook, body, CTA)
   - Recommend campaign structure
   - Output to GTM_PLAN.md

3. EMAIL SEQUENCES (for Instantly)
   - Write 3-4 email sequence
   - Use variables: {{firstName}}, {{companyName}}, {{lastName}}
   - Include subject lines and body copy
   - Output to GTM_PLAN.md

4. CALL SCRIPTS
   - Write cold call opener and full script
   - Include objection handling
   - Define lead list criteria
   - Generate sample CSV for GHL import
   - Output to GTM_PLAN.md
